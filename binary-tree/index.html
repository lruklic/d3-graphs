Binary Search Tree
data structure

A data structure in which a record is linked to two successor records, referred to as the left branch when greater and the right when less than the previous record.

Duplicates are usually not supported in binary trees.

Sorting
algorithms

Algorithm that puts elements of a list in a certain order.

There are many different sorting algorithms that differ on speed, complexity, memory usage etc.

Bubble sort
Exchanging

The algorithm starts at the beginning of the data set. It compares the first two elements, and if the first is greater than the second, it swaps them. It continues doing this for each pair of adjacent elements to the end of the data set. It then starts again with the first two elements, repeating until no swaps have occurred on the last pass.

Best time: n

Worst time: n^2

Average time: n^2
